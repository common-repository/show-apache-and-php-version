Show Apache and PHP version

== Description ==

Displays Apache and PHP version in footer of admin section for easy access.

These information are usually needed if reporting bugs or checking hosting service configuration. Wordpress displays only it's own version which is not enough.

Plugin is very simple and uses almost no resources. After all, it just displays version information.

== Installation ==

1. Extract package and upload to plugins directory
2. Activate the plugin through the 'Plugins' menu in WordPress
